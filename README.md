# Ruboty::Redis
Store [Ruboty](https://github.com/r7kamura/ruboty/)'s memory in Redis.

## Usage
```ruby
# Gemfile
gem "ruboty-redis"
```

## ENV
```
REDIS_URL           - Redis URL (e.g. redis://foo:bar@example.com:6379/)
REDIS_SAVE_INTERVAL - Interval sec to push data to Redis (default: 5)
```
